# KeyGo UI — Esqueleto

Proyecto base (sin funcionalidades) creado según `FRONTEND_DEVELOPER_GUIDE.md`.

Requisitos:
- Node 18+
- npm o pnpm

Instalación local:

```bash
npm install
npm run dev
```

Variables de entorno recomendadas (ejemplo `.env.local`):

```
VITE_KEYGO_BASE=http://localhost:8080/keygo-server
VITE_TENANT_SLUG=keygo
VITE_CLIENT_ID=keygo-ui
VITE_REDIRECT_URI=http://localhost:5173/callback
```
